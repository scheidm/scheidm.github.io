Tools of the Trade

While I was always attracted to Vim for the speed of expression and wealth of plugins, the proliferation of free or inexpensive server options has made this command line tool even more valuable. It is the standard by which I judge other programs in my dev environment. Once I found Vundle and could host my whole setup on a GitHub repository, bridging my configuration between computers has been a breeze

I still remain loyal to Bash, and Tmux has made managing multiple sessions so much simpler and more powerful than the old days with Screen. Vim-like bindings make the tool instantly comfortable, and make quick work of checking on server errors in my latest web-app.
