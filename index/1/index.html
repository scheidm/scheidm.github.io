Who am I?

Greetings visitor!

My name is Mark Scheid, a programmer hailing from Northboro, MA. I have a passion for creation, whether it be a new web application, a piece of fiction, a community event, or the perfect music playlist. My passion lies in the possibilities unlocked by modern web technologies, and I love learning new tools and applying them to solving interesting problems. I created this website to showcase my work, in the hopes of finding like-minded people to collaborate with.

Ruby has been my primary production language since April 2013. I love the freedom of expression Ruby gives you, and the quality of the libraries available on Rubygems. I likewise have had a lot of opportunity to use Python, both for all those 1-off command line tools you end up needing, and for academic work.

Over the last year, the vast majority of my time has been going towards a web application called Quest Till Done. Built on Rails, it's a very visual productivity tool, and so most of our interface has been JavaScript heavy. For the last few months, I've been playing with frameworks as we begin to migrate the whole project to an AJAX approach.

Most of my academic programming in the last few years has been Java, so I remain very comfortable with that language. My first formal code education was in C++, and I remain comfortable enough that I can drop some C extensions into my code when necessary."
